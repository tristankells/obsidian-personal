# Old Way (SMART)
- SMART (Specific, Measurable, Achievable, Relevant, Time Bound)
- Very rigid, the "Time Bound" constraint can encourage procrastination... "I have until then to do it, I can start tomorrow"
# New Way (Outcome + Output)
- **Outcome**. Want to happen. Can be very vague now, this is more natural; 
	- "I want to lose weight"
	- "I want to learn guitar" 
	- "I want to practice Singhalese"
	- "I want to be a principal engineer"
- **Output**. What will you do. Must be a regular, repeatable action that helps you achieve your outcome. Can have multiple outputs per outcome.
	- Why this method? 
		- Incorporates a consistent feedback loop
			![[Pasted image 20250916170027.png]]
		- Encourages your to evaluate the output consistently in relationship to your outcome.
		- More actions (outputs) == more feedback cycles == more learning + progress.
		- Outcome is not time limited.
- **Tracking**. To ensure your outputs are helping you achieve your outcome, you need a way of tracking BOTH.
	![[Pasted image 20250916171356.png]]
# Examples
- Outcome -> "I want bigger muscles"
- Output -> "I will go lift heavy weights 4 times a week"
- Outcome -> "Get healthy and lose weight"
- Output -> "I will weigh myself everyday"
	- "Stick to a set basic diet"
- Outcome -> "Get better at piano"
- Output -> "Learn one song every week"
- Outcome -> "Learn medicine for finals"
- Output -> "Will learn one topic per day"
